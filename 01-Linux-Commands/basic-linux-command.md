# Basic Linux Commands

## 1. Navigation Commands
- pwd – Print working directory
- ls – List files and folders
- cd – Change directory
- tree – View directory structure

## 2. File & Directory Commands
- touch filename – Create an empty file
- mkdir foldername – Create folder
- cp source destination – Copy file or folder
- mv oldname newname – Move or rename
- rm filename – Delete a file
- rmdir foldername – Delete empty folder
- rm -r foldername – Delete folder with files

## 3. Permissions & Ownership
- chmod 755 filename – Change permissions
- chown user:group filename – Change ownership

## 4. System Information
- uname -a – System details
- whoami – Logged-in username
- df -h – Disk usage
- free -m – RAM info
