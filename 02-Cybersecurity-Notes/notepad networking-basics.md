## Linux File Permissions

Linux uses file permissions to control access to files and directories.

- r (read): view file content
- w (write): modify file
- x (execute): run file as program

Permission format:
-rwxrwxrwx

Hidden Files:
Files starting with '.' are hidden.
Example: .bashrc, .secret_file

Hidden files may store sensitive data.
# Networking Basics – Day 1

## 1. IP Address
An IP address uniquely identifies a device on a network. It can be IPv4 or IPv6.

## 2. Default Gateway
The default gateway connects the local network to external networks like the internet.

## 3. Ping Command
Ping is used to test connectivity between two systems and measure network latency.

## Commands Used
- ip a
- ip route
- ping google.com
