# Network Reconnaissance

## Description
This folder contains notes, commands, and screenshots related to
network reconnaissance techniques used in cybersecurity and penetration testing.

## Topics Covered
- Passive Reconnaissance
- Active Reconnaissance
- Google Dorking
- Port Scanning
- Ping Sweeps
- Traceroute

## Tools Used
- Nmap
- Netcat
- Wireshark
- Whois

## Learning Outcome
Understood how attackers gather information about targets and how
defenders can reduce attack surface exposure.
