---

## Why This Matters in Cybersecurity

Networking is the backbone of cybersecurity.  
Before any attack or defense, understanding how data flows across a network is essential.

- Misconfigured networks can expose sensitive systems
- Weak file permissions can lead to data leakage or privilege escalation
- Reconnaissance is the first step in most cyber attacks

---

## Real-World Security Insight

Many real-world breaches happen not because of advanced hacking, but due to:
- Open ports left exposed
- Incorrect file permissions
- Poor network visibility
- Unmonitored DNS or routing issues

Learning these basics helps in identifying and preventing such vulnerabilities early.
