1️.Authentication
Authentication is the process of verifying the identity of a user.
👉 It answers the question:
“Who are you?”
🔹 Examples of Authentication:
Username and Password
OTP sent to mobile/email
Fingerprint / Face ID
Smart card / ATM card + PIN
🔹 Where Authentication is Used:
Gmail login
ATM machines
College student portal
Social media accounts
Online banking
🔹 Types of Authentication:
Single-Factor Authentication (SFA): Password only
Two-Factor Authentication (2FA): Password + OTP
Multi-Factor Authentication (MFA): Password + OTP + Biometric

2️. Authorization
Authorization is the process of deciding what resources a user is allowed to access.
👉 It answers the question:
“What are you allowed to do?”
🔹 Examples of Authorization:
Student can view marks but cannot edit them
Employee can access company files but not admin settings
User can read data but cannot delete it
🔹 Where Authorization is Used:
File permissions (read/write/delete)
Role-based access in companies
Admin vs user privileges
Cloud systems and databases

3 . Real-Life Example (ATM)
Authentication:
Inserting ATM card + entering PIN
Authorization:
Withdrawing money, checking balance, printing receipt 

4 . Importance in Cybersecurity
Prevents unauthorized access
Protects sensitive data
Reduces hacking and data breaches
Essential for secure systems and networks

5 . Cybersecurity Use Cases
Web application security
Network access control
Cloud security (AWS, Azure)
Identity and Access Management (IAM)
Banking and financial systems

🔑 Summary
Authentication = Identity verification
Authorization = Permission control
Authentication always comes before Authorization
Both are critical pillars of Cybersecurity

Authentication plays a major rolr in Security aspects
